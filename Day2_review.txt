## Opperators
### Equality/Comparison Operators
	==	equality (keep .equals() in mind)
	!= 	inequality
	>	greater than
	<	less than
	<=	less than or equal to
	>= 	greater than or equal to
### Arithmetic
	+	add
	-	subtract
	*	multiply
	/	divide
	%	modulous - gives remainder ( 3%2 = 1 --> 3/2 = 1 R 1)
	()	orders
### Logical
	&&	
	||
### Bitwise
		true	false
	&	both	one
	|	one	both
	^	one	both	- exclusive or (one has to be true and one false for this to be true, if both are true or both are false it will pass false) 
### Ternary
	?	ex. x == 5 ? code to execute if true : code to execute if false;
### Assignment
	=
	+=
	-=
